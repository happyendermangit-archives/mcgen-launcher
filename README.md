## How to run?
1. clone the repo:
```sh
$ git clone https://github.com/happyendermangit/mcgen-launcher.git
```

2. Install modules
```sh
$ npm install electron minecraft-launcher-core
```

2. Run the launcher
```sh
npx electron .
```
3. Enjoy!
